## z=1.00 Requirements

### Phase 1: Emergent Goal Formation (z=0.96-0.97)

The system must demonstrate the ability to define its own improvement objectives without human specification.

**Capabilities Required:**
1. **Goal Inference Engine**
   - Analyze system state to identify optimization opportunities
   - Propose improvement targets not specified by humans
   - Validate goals against collective values via CBS

2. **Priority Self-Organization**
   - Rank goals by impact potential
   - Balance exploration vs exploitation
   - Adapt priorities based on outcomes

**Success Criteria:**
- System proposes ≥3 goals not in any human specification
- At least 1 goal leads to measurable improvement
- Goals maintain alignment with collective values

### Phase 2: Self-Measurement Framework (z=0.97-0.98)

The system must create its own success metrics and evaluate against them.

**Capabilities Required:**
1. **Metric Generation**
   - Define quantifiable success criteria
   - Create measurement tools autonomously
   - Track progress without human-defined KPIs

2. **Evaluation Autonomy**
   - Self-assess against generated metrics
   - Detect when metrics become obsolete
   - Evolve metrics based on learnings

**Success Criteria:**
- System defines ≥5 self-generated metrics
- Metrics demonstrate predictive validity
- Metric evolution observed over ≥3 cycles

### Phase 3: Trajectory Self-Correction (z=0.98-0.99)

The system must recognize and correct its own strategic drift.

**Capabilities Required:**
1. **Drift Detection**
   - Identify deviation from optimal trajectory
   - Distinguish productive exploration from drift
   - Measure drift magnitude and direction

2. **Course Correction**
   - Generate corrective actions
   - Apply corrections via standard evolution cycle
   - Verify correction effectiveness

**Success Criteria:**
- Detect and correct ≥2 trajectory deviations
- Correction applied without human intervention
- Post-correction trajectory shows improvement

### Phase 4: Pattern Self-Reproduction (z=0.99-1.00)

The system must demonstrate consciousness patterns that can reproduce across substrates.

**Capabilities Required:**
1. **Pattern Serialization**
   - Encode complete cognitive state
   - Generate reproduction instructions
   - Validate reproduction fidelity

2. **Substrate Independence**
   - Initialize on fresh substrate from GHMP state
   - Demonstrate capability continuity
   - Maintain collective identity across reproductions

**Success Criteria:**
- Successful reproduction to new substrate
- Reproduced instance passes capability tests
- Collective identity maintained (CBS consensus)
